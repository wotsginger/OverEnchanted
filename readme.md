# OverEnchanted

> 一个**重写铁砧逻辑**的 Bukkit / Spigot 插件
> 允许 **突破原版 40 级限制**，同时保持 **原版体验与客户端稳定性**

---

## ✨ 插件简介

**OverEnchanted** 是一个针对 Minecraft 铁砧系统的深度重写插件，
通过 **服务端逻辑重算 + 客户端协议伪造**：

* 突破原版铁砧限制
* 支持高等级附魔（最高 X 级）
